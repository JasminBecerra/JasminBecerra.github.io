I made the map below in my Intro to Cartography course during the Fall of 2015.
Inspired by Fix Wiki Maps, this assignment asked that each student search through wikipedia pages with data but no visual graphics. I ended up finding a page with Tangerine and citrus fruit production (the best type of fruit if you ask me!) that included an index of the largest 20 producers.
With this information, I created the following proportional symbols map via ArcMap to represent the Tangerine production data.
