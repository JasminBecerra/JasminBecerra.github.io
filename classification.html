The following is an overview of a remote sensing assignment I completed that involved classifying land cover using two different supervised classification methods-- Maximum Likelihood, and Support Vector Machine. Supervised classification requires that the user/analyst provide a set of training sites that represent a sample of the classes they are interested in mapping, where as unsupervised classification software generates classes with no sample input from the user.

For this assignment, I focused on classifying land cover in the greater Madison, WI area. Below are some snapshots of my process:
